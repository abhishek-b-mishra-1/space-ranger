# Space Ranger 🚀

A 2D side-scrolling space shooter built with Python and pygame. Fight through 5 levels of increasingly brutal enemies, face unique bosses, and survive to the end.

## Play in Browser

👉 **[Play Now](https://abhishek-b-mishra-1.github.io/space-ranger)**

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrow Keys | Move |
| SPACE / Left Click | Shoot |
| B | Deploy Bomb |
| ESC | Pause |
| R | Restart (Game Over) |

## Features

- 5 unique boss fights (Asteroid Prime → Alien Cruiser → Mecha Titan → Leviathan → Dreadnought)
- 3 weapon types: Laser, Fire, Magnetic (unlocked at Level 3)
- Bomb system — clears the screen instantly
- Shadow clone power-up
- Positive & negative power-ups
- Procedurally generated music and sound effects
- Parallax scrolling backgrounds with level themes

## Run Locally

```bash
pip install pygame
python main.py
```

## Requirements

- Python 3.10+
- pygame 2.0+
